# 2. LA V1 DOIT ÊTRE UNE BASE SOLIDE

La V1.1 ne doit pas être construite sur une base instable.

Avant d'ajouter des améliorations, tu dois corriger :

* Les bugs existants.
* Les erreurs de logique.
* Les routes cassées.
* Les liens incorrects.
* Les formulaires incomplets.
* Les erreurs de permissions.
* Les problèmes d'authentification.
* Les problèmes d'isolation entre établissements.

# 3. RÈGLE ABSOLUE : NE RIEN CASSER

Pendant le développement de la V1.1 :

* Ne pas casser les fonctionnalités fonctionnelles de la V1.
* Ne pas supprimer une fonctionnalité existante sans justification.
* Ne pas modifier inutilement l'architecture.
* Ne pas modifier les modèles de données sans vérifier les conséquences.
* Ne pas casser les permissions.
* Ne pas casser l'authentification.
* Ne pas casser l'isolation des écoles.
* Ne pas casser les routes existantes.
* Ne pas créer de régression.

Chaque modification doit être testée.

La stabilité est prioritaire sur l'ajout de nouvelles fonctionnalités.

# 4. SCRIPT DE DONNÉES DE TEST V1

Créer un script dédié pour générer des données de test réalistes pour la V1.

Le script doit pouvoir créer automatiquement un environnement de test complet comprenant :

* Une plateforme.
* Plusieurs écoles.
* Des utilisateurs.
* Les différents rôles.
* Des comptes administrateurs.
* Des enseignants.
* Des secrétaires.
* Des comptables.
* Des données de démonstration nécessaires au fonctionnement de la V1.

Le script doit être :

* Reproductible.
* Facile à exécuter.
* Documenté.
* Sécurisé.
* Utilisable uniquement pour l'environnement de développement ou de test.

Créer par exemple un fichier du type :

`seed_test_data.py`

ou utiliser la structure appropriée au framework utilisé.

Le script doit permettre de tester facilement :

* La connexion.
* Les différents rôles.
* Les permissions.
* L'isolation des écoles.
* Les tableaux de bord.
* Les principales fonctionnalités de la V1.

Ne jamais utiliser de vraies données personnelles.
